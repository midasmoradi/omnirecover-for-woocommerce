# Security Policy

## Reporting

Report vulnerabilities privately — do not open public issues for security-sensitive findings.

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Notes

Messaging API keys are stored in WordPress options. Restrict admin access and use environment-appropriate secrets rotation for Twilio, UltraMsg, and Telegram credentials.
